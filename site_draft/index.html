SYLLABUS

14-Week Curriculum Overview

This course combines Python programming fundamentals with web development skills, applying computational thinking to humanities research and digital scholarship.

Python Fundamentals

Weeks 1-7

  • Programming basics & setup
  • Variables, data types, control structures
  • Functions, modules, data structures
  • File I/O, text processing
  • Web scraping & APIs

Web Development

Weeks 8-14

  • HTML foundations & semantic markup
  • CSS styling & responsive design
  • JavaScript fundamentals
  • DOM manipulation & interactivity
  • Final portfolio presentation

ASSIGNMENTS

Weekly Coding Exercises

ONGOING

Weekly hands-on coding assignments designed to reinforce concepts from each lesson. Includes debugging challenges, creative coding problems, and peer code review.

200 POINTS PRACTICE

Text Analysis Project

DUE: WEEK 6

Write a Python program to analyze a literary text of your choice. Extract patterns, word frequencies, and sentiment analysis using computational methods.

150 POINTS PROJECT

Digital Archive Creation

DUE: WEEK 10

Build a responsive digital archive for historical documents or cultural artifacts. Include metadata, search functionality, and accessible user interface design.

175 POINTS WEB PROJECT

Interactive Research Tool

DUE: WEEK 12

Develop an interactive web application that serves humanities research. Could be a timeline, data visualization, annotation tool, or digital exhibit.

150 POINTS INTERACTIVE

Digital Humanities Capstone

DUE: WEEK 14

Design and implement a comprehensive digital humanities project that combines coding skills with humanistic inquiry. Present your research, methodology, and findings.

225 POINTS CAPSTONE

Digital Portfolio

DUE: WEEK 14

Create a professional digital portfolio showcasing your coding journey, projects, and reflections. Must include responsive design and demonstrate technical skills learned.

100 POINTS PORTFOLIO
