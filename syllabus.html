DETAILED SYLLABUS

Fall 2025 • MWF 10:00-10:50 AM • Beasley Hall

Course Description

This course introduces programming for students in the Humanities and Digital Culture and Data Analytics (DCDA) Minor. Students learn Python basics as they apply to intellectual, cultural, and creative work central to the Humanities. We explore how computational analysis can broaden our ability to analyze texts of significance and practice critical analysis of humanistic work.

Learning Outcomes

  • Use computer programming as an exploratory research tool for cultural questions
  • Reflect on limitations of computational methods in Digital Humanities
  • Develop basic proficiency with Python programming language
  • Analyze rhetorical acts and texts in various media
  • Use humanistic modes of inquiry across space and time

Key Course Features

  • Un-grading System: Focus on learning through self-reflection essays
  • AI Encouraged: Use of ChatGPT, Gemini, etc. with documentation
  • Google Colab: Browser-based Python coding environment
  • Cultural Analytics: Apply coding to humanities research questions

Python Fundamentals (Weeks 1-7)

WEEK 1: Introduction & Setup

Topics: Course overview, computational thinking, Python installation, IDE setup

Activities: Environment configuration, first "Hello World" program

Assignment: Setup verification & reflection essay

WEEK 2: Variables & Data Types

Topics: Integers, floats, strings, booleans, type conversion

Activities: Interactive coding exercises, debugging practice

Assignment: Personal data calculator project

WEEK 3: Control Structures

Topics: If/else statements, loops (for, while), logical operators

Activities: Decision tree programming, iteration challenges

Assignment: Interactive story generator

WEEK 4: Functions & Modules

Topics: Function definition, parameters, return values, importing libraries

Activities: Code organization workshop, library exploration

Assignment: Humanities toolkit module creation

WEEK 5: Data Structures

Topics: Lists, dictionaries, tuples, sets, data manipulation

Activities: Database simulation, collection management

Assignment: Digital collection organizer

WEEK 6: File I/O & Text Processing

Topics: Reading/writing files, text analysis, regular expressions

Activities: Literary text parsing, pattern recognition

Assignment: Text Analysis Project (Due)

WEEK 7: Web Scraping & APIs

Topics: HTTP requests, Beautiful Soup, API integration, data ethics

Activities: Museum database exploration, ethical scraping discussion

Assignment: Research data collection script

Web Development (Weeks 8-14)

WEEK 8: HTML Foundations

Topics: HTML structure, semantic markup, forms, accessibility basics

Activities: Building scholarly article markup, form creation

Assignment: Digital CV/portfolio HTML framework

WEEK 9: CSS Styling & Layout

Topics: CSS selectors, box model, flexbox, grid layout

Activities: Typography workshop, layout design challenges

Assignment: Styled academic presentation page

WEEK 10: Responsive Design

Topics: Media queries, mobile-first design, CSS frameworks

Activities: Multi-device testing, accessibility auditing

Assignment: Digital Archive Creation (Due)

WEEK 11: JavaScript Basics

Topics: Variables, functions, events, debugging tools

Activities: Interactive storytelling elements, user interaction design

Assignment: Dynamic content showcase

WEEK 12: DOM Manipulation

Topics: Document object model, element selection, dynamic content

Activities: Interactive timeline creation, data visualization

Assignment: Research presentation tool

WEEK 13: Interactive Web Apps

Topics: APIs, AJAX, local storage, deployment basics

Activities: Capstone project development, peer code review

Assignment: Final project preparation

WEEK 14: Portfolio Presentation

Topics: Project documentation, presentation skills, reflection

Activities: Capstone presentations, peer evaluation, course reflection

Assignment: Digital Humanities Capstone (Due)
